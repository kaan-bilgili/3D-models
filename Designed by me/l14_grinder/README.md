# Herb Grinder Project

This project contains a **functional grinder model** designed in Fusion 360.  
The purpose is both **educational** and **practical**, created during free time practice.

## Files
- `l14_grinder/source/grinder.f3d` → Fusion 360 source file  
- `l14_grinder/stl/grinder.stl` → 3D printable grinder model  

## Notes
- Designed as a **functional grinder** with interlocking parts.  
- Created to practice **Fusion 360 modeling techniques** such as shells, extrusions, and circular patterns for the teeth.  
- Dimensions may need tolerance adjustments depending on the printer.  
- Recommended to print with durable materials (e.g., PETG or ABS) for strength.
