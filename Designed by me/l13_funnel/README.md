# Funnel Project

This project contains a simple **funnel model** designed in Fusion 360.  
The purpose is **educational**, focusing on using key modeling tools like **Shell** and **Loft**.

## Files
- `l13_funnel/source/funnel.f3d` → Fusion 360 source file  
- `l13_funnel/stl/funnel.stl` → 3D printable funnel model  

## Notes
- Designed to practice and demonstrate:  
  - **Loft** → to create the smooth transition from wide opening to narrow tube  
  - **Shell** → to hollow the solid body into a thin-walled funnel
